# Parallax images

Place **7 JPEG or WebP** files in this folder using these exact names:

| File | Slot | Recommended aspect ratio | Min. size |
|------|------|------------------------|-----------|
| `01-center.jpg` | Center hero | **1:1** (square) | 1200 × 1200 px |
| `02-top-left.jpg` | Top-left | **7:6** (slightly wide) | 1400 × 1200 px |
| `03-top-right-portrait.jpg` | Top-right tall | **4:9** (tall portrait) | 800 × 1800 px |
| `04-mid-right.jpg` | Mid-right | **1:1** (square) | 1200 × 1200 px |
| `05-bottom-left.jpg` | Bottom-left | **4:5** (portrait) | 1000 × 1250 px |
| `06-bottom-right.jpg` | Bottom-right | **6:5** (slightly wide) | 1200 × 1000 px |
| `07-accent.jpg` | Small accent | **1:1** (square) | 800 × 800 px |

Images are cropped with `object-cover` inside fixed aspect-ratio frames, so matching these ratios avoids important content being clipped. Slightly wider or taller source images are fine.

**Subject tips:** mix wide shots (panels, job sites) with tight detail shots (tools, wiring, EV chargers) for depth in the zoom effect.
